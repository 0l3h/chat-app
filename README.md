# Chat app

**Tech stack**:
- React.js
- Nest.js
- Socket.io
- Tailwind.css
- PostgreSQL
- Prisma
- Railway (hosting)
- Vite

##Running locally##

Open client folder and run `npm run start`. Then, open server folder and run 
```
npm i
npx prisma generate
npx prisma migrate
```
After which you can start a server using `npm run start:dev`.

Project URL: https://chat-app-production-2301.up.railway.app/
